Atividade nº11 | Quiz

Atividade em dupla, crie um quiz com as seguintes especificações:
1 - Perguntas e Alternativas:
1.1 - O quiz deve ter no mínimo 5 perguntas.
1.2 - Cada pergunta deve ter 4 alternativas de resposta.
2 - Interação com o Usuário:
2.1 - Solicite ao usuário que escolha uma alternativa para cada pergunta.
3 - Resultado do Quiz:
3.1 - Após o usuário responder todas as perguntas, exiba um resumo com:
3.2 - A alternativa correta para cada pergunta.
3.3 - A alternativa que o usuário escolheu para cada pergunta.
3.4 - A soma total de acertos (quantidade de respostas corretas).

Atividade nº15 | Acerte o numero Correto

Crie um programa em JavaScript que:
1. Armazene um número fixo entre 1 e 10 (escolhido por você).
2. Solicite ao usuário que tente adivinhar o número.
3. Enquanto o usuário não acertar o número, o programa deve continuar solicitando novas tentativas.
4. Exiba uma mensagem para o usuário tentando novamente se o palpite estiver errado.
5. Exiba uma mensagem de sucesso quando o usuário acertar o número.
